Colorizor

The best syntax highlighter you'll find on the web! It makes it easy to integrate into your websites, by automatically loading the correct files for each code block. It allows you to select the theme you want when calling the API with the script tag and loaded it automatically for you.

Just by selecting any code with this API with automatically select the code for the viewers.

function HeaderHeight(header) {
  if (screen.width < 480) {
    header = 150;
  } else if (screen.width >= 480 && screen.width < 720) {
    header = 250;
  } else if (screen.width >= 720 && screen.width < 1024) {
    header = 250;
  } else {
    header = 400;
  }
}

Features

- Auto Language Detection
- Auto Theme Detection
- Auto Code Selection
- Inline Block Code
- In-Text Code

There are many features that aren't listed. Features suck as code blocks without any style and telling the code block not to have any background color.

Usage

URL:
https://colorizor.github.io/colorizor.js ?theme=...

Example:
<script type="text/javascript" src="https://colorizor.github.io/colorizor.js?theme=Salmon" async></script>

Define Language:
<code language="cpp">...</code>

Example:
<pre><code language="cpp">...</code></pre>
