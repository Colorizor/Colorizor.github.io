Colorizor

The best syntax highlighter you'll find on the web! It makes it easy to integrate into your websites, by automatically loading the correct files for each code block. It allows you to select the theme you want when calling the API with the script tag and load it automatically for you.

Clicking any code block will automatically select all the code for the viewer.

Features

- Auto Language Detection
- Auto Theme Detection
- Auto Code Selection
- Inline Block Code
- In-Text Code

There are many features that aren't listed. Features suck as code blocks without any style and telling the code block not to have any background color.

Usage

URL:
https://colorizor.github.io/colorizor.js ?theme=...

Example:
<script type="text/javascript" src="https://colorizor.github.io/colorizor.js?theme=Salmon" async></script>


Define Language:
<code language="cpp">...</code>

Example:
<pre><code language="cpp">...</code></pre>


No Language:
<code language="none">...</code>

Example:
<pre><code language="none">...</code></pre>


In-Text Code:
<span class="none">...</span>

Background:
With
<pre>...</pre>
or
<pre background="true">...</span>
or Without
<pre background="false">...</span>
