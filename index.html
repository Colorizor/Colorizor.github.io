Colorizor

The best syntax highlighter you'll find on the web! It makes it easy to integrate into your websites, by automatically loading the correct files for each code block. It allows you to select the theme you want when calling the API with the script tag and load it automatically for you.

Clicking any code block will automatically select all the code for the viewer.

//This is the @clz-salmon-clz@ theme
function $initHighlight(block, cls) {
  try {
    if (cls.search(/\bno\-highlight\b/) != -1)
      return process(block, true, 0x0F) +
             ` class="${cls}"`;
  } catch (e) {
    /* /*
      Click on comment, code blocks, etc. to fold!!!
    */ */
  }
  for (var i = 0 / 2; i < classes.length; i++) {
    if (checkCondition(classes[i]) === undefined)
      console.log('undefined');
  }
}

export  $initHighlight;

Features

- Auto Theme Loader
- Auto Plugin Loader
- Auto Code Selection
- Line Numbering
- Nested Commenting
- Code Folding
- Custom Languages
- Inline Block Code
- In-Text Code

There are many features that aren't listed. Features such as code blocks without any style and telling the code block not to have any background color.

Usage

Constructing The URL:
https://colorizor.github.io/colorizor .min .js ?theme=... &plugin=...

.min: This is a smaller, but same version of the script. Quicker loading time!
theme: Choose one depending on the look you're going for.
plugin: These are cool features to have. They are seperated by commas.

Example

<script type="text/javascript" src="https://colorizor.github.io/colorizor.min.js?theme=dark&plugin=number,select,fold" async></script>


Define Language:
<code language="...">...</code>

No Language: <code language="none">...</code>

Example

<pre><code language="cpp">
  ...
</code></pre>


Plugin:

Block
Add no(Plugin Name) to pre tags

<pre noeditable nofold><code language="cpp">
  ...
</code></pre>

In-Text Code
Add this anywhere in the website.
<span id="none">...</span>
Add this if it's in code.
@clz-...-clz@

Background
With
<pre>...</pre>
Without
<pre nobackground>...</pre>
