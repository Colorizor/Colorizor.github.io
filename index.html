Colorizor

The best syntax highlighter you'll find on the web! It makes it easy to integrate into your websites, by automatically loading the correct files for each code block. It allows you to select the theme you want when calling the API with the script tag and load it automatically for you.

Clicking any code block will automatically select all the code for the viewer.

function $initHighlight(block, cls) {
  try {
    if (cls.search(/\bno\-highlight\b/) != -1)
      return process(block, true, 0x0F) +
             ` class="${cls}"`;
  } catch (e) {
    /* 
      Tap on the comment to collapse!!!
      URL Plugin: [Colorizor URL Support][https://colorizor.github.io/Plugins]
      /*
        Nested Comment for all languages
      */
    */
  }
  for (var i = 0 / 2; i < classes.length; i++) {
    if (checkCondition(classes[i]) === undefined)
      console.log('undefined');
  }
}

export  $initHighlight;

Features

- Auto Language Detection
- Auto Theme Detection
- Auto Code Selection
- Line Numbering
- Nested Commenting
- Code Folding
- Custom Languages
- Inline Block Code
- In-Text Code

There are many features that aren't listed. Features such as code blocks without any style and telling the code block not to have any background color.

Usage

URL:
https://colorizor.github.io/colorizor .min .js ?theme=... &plugin=...

Example:
<script type="text/javascript" src="https://colorizor.github.io/colorizor.min.js?theme=Salmon&plugin=number,select" async></script>


Define Language:
<code language="...">...</code>

Example:
<pre><code language="cpp">...</code></pre>


No Language:
<code language="none">...</code>

Example:
<pre><code language="none">...</code></pre>


Block Plugin:
Add no(Plugin Name) to pre tags
<pre no...>...</pre>

In-Text Code:
<span id="none">...</span>

Background:
With
<pre>...</pre>
Without
<pre nobackground>...</span>
